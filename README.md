# Troch [![Build Status](https://secure.travis-ci.org/kano4/troch.png)](http://travis-ci.org/kano4/troch)
Troch is a website monitoring tool.

# Requirements
- Ruby 1.9.2
- crontab
- OpenSSL
- Redis

# Installation
    git clone https://github.com/kano4/troch.git
    cd troch
    bundle
    rake db:setup
    rails s
